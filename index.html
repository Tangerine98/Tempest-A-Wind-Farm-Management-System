A Windfarm Management System

The software will be useful for wind turbine owners, operators and asset managers to capture the full potential of wind energy projects. It's a modern and user-friendly software solution to save time, improve portfolio performance, increase knowledge and enable proactive wind-farm management. Wind turbines and other devices are connected to Tempest via on-site SCADA systems- resulting in a single powerful tool to monitor, analyse, plan and control diverse wind energy portfolios of any size.

The software is made using:-
1. Front End
    - Basics : HTML, CSS
    - CSS Tools : Materialize CSS, Bootstrap
    - JS Framework : React JS with Redux
2. Back End
    - Scripting Language : Django Python
    - Database : MySQL, Redis for caching, 3nf Normalization
3. Web Platforms : Digital Ocean
4. Vizualisation : D3.js
